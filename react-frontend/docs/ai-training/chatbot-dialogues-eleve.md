# Dialogues Chatbot - Élèves

## Vue d'ensemble

Ce document contient des exemples de dialogues entre élèves et le chatbot EdConnekt, basés sur les fonctionnalités identifiées dans le code source.

## Intentions et Réponses

### Consultation des notes

**Exemples de phrases utilisateur :**
- "Comment voir mes notes ?"
- "Où sont mes résultats du dernier trimestre ?"
- "Je veux consulter mes évaluations"
- "Comment accéder à mes notes ?"
- "Comment voir mes moyennes par matière ?"
- "Où trouver mes résultats de compétences ?"
- "Comment comprendre les différents types d'évaluation ?"
- "Comment voir mes notes continues ?"
- "Comment voir mes notes d'intégration ?"
- "Comment voir mes notes trimestrielles ?"
- "Comment analyser ma progression ?"
- "Comment voir mes moyennes par domaine ?"
- "Comment télécharger mes rapports de notes ?"
- "Comment comparer mes résultats ?"
- "Comment voir mes points forts et faibles ?"
- "Comment comprendre les pourcentages ?"
- "Comment voir l'historique de mes notes ?"

**Réponses attendues du chatbot :**
- "Pour consulter vos notes, allez dans la section 'Mes Notes' depuis le menu principal. Vous y trouverez vos résultats par matière avec des graphiques de progression."
- "Vos notes sont disponibles dans l'onglet 'Mes Notes'. Vous pouvez sélectionner la période d'évaluation (trimestre, mois) et consulter vos résultats par compétence."
- "Accédez à 'Mes Notes' dans le menu pour voir vos évaluations. Vous pouvez analyser votre progression et identifier les points d'amélioration."
- "Dans 'Mes Notes', vous pouvez voir vos moyennes par matière, vos résultats par compétence et télécharger vos rapports d'évaluation."
- "Trois types d'évaluation sont affichés : Continue (évaluations régulières), Intégration (évaluations de synthèse), Trimestrielle (bilans périodiques)."
- "Les notes continues montrent votre progression régulière tout au long de l'année sur chaque compétence."
- "Les notes d'intégration évaluent votre maîtrise globale des compétences acquises dans une matière."
- "Les notes trimestrielles fournissent un bilan complet avec moyennes et recommandations pour progresser."
- "Analysez votre progression via les graphiques qui montrent l'évolution de vos compétences dans le temps."
- "Vos moyennes sont calculées par domaine (Langues, Sciences, etc.) et par matière pour un suivi complet."
- "Téléchargez vos rapports de notes en PDF depuis la barre d'outils pour les conserver ou les partager."
- "Comparez vos résultats entre les différents types d'évaluation pour identifier vos forces et axes d'amélioration."
- "Vos points forts et faibles sont identifiés automatiquement avec des indicateurs visuels (vert = acquis, orange = en cours, rouge = à travailler)."
- "Les pourcentages indiquent votre niveau d'acquisition de chaque compétence : au-dessus de 70% = acquis, en dessous = à renforcer."
- "L'historique de vos notes est conservé pour vous permettre de suivre votre progression sur l'année entière."

**Réponses attendues du chatbot :**

- "Allez dans 'Mes Cours', choisissez une matière, puis consultez les leçons avec leurs ressources (PDF, vidéos, exercices)."
- "Dans 'Mes Cours', vous pouvez marquer les leçons comme consultées et télécharger les supports de cours."
- "Cliquez sur un cours dans 'Mes Cours' pour voir les détails, les leçons et les ressources disponibles."

### Ressources pédagogiques

**Exemples de phrases utilisateur :**
- "Où trouver des ressources pour réviser ?"
- "Comment télécharger des documents ?"
- "Y a-t-il des exercices disponibles ?"
- "Je cherche des supports de cours"
- "Comment accéder aux ressources de mes cours ?"
- "Où trouver des vidéos explicatives ?"
- "Comment rechercher une ressource spécifique ?"
- "Où voir les ressources par matière ?"
- "Comment filtrer les ressources ?"
- "Où voir les ressources payantes vs gratuites ?"
- "Comment comprendre les types de ressources ?"
- "Où voir les ressources les plus récentes ?"
- "Comment voir les détails d'une ressource ?"
- "Où voir les ressources partagées par l'enseignant ?"
- "Comment utiliser les ressources pour mes devoirs ?"

**Réponses attendues du chatbot :**
- "Les ressources sont disponibles dans la section 'Ressources' du menu. Vous y trouverez des documents, exercices et supports de cours."
- "Dans 'Ressources', vous pouvez rechercher et télécharger des documents par matière et par type de contenu."
- "La section 'Ressources' contient des supports pédagogiques organisés par matière et niveau de difficulté."
- "Accédez aux ressources via 'Ressources' ou directement depuis vos cours. Vous pouvez télécharger PDF, vidéos et exercices."
- "Utilisez la barre de recherche pour trouver rapidement une ressource par titre, description ou compétence."
- "Filtrez les ressources par matière (français, mathématiques, etc.), type de fichier (PDF, vidéo) ou statut (gratuit/payant)."
- "Les ressources sont organisées par domaines : Langues et Communication, Sciences Humaines, STEM, et Créativité Artistique/Sportive."
- "Les ressources payantes sont marquées d'un badge jaune 'Payant' et nécessitent un paiement pour être consultées."
- "Les types de ressources incluent PDF, documents Word, PowerPoint, vidéos, images et liens externes."
- "Les ressources les plus récentes apparaissent en premier dans la liste, avec la date d'ajout visible."
- "Cliquez sur une ressource pour voir ses détails complets : description, auteur, taille, version et ressources associées."
- "Les enseignants partagent des ressources avec visibilité 'Classe' que vous pouvez consulter selon vos droits d'accès."
- "Utilisez les ressources pour vos devoirs, révisions ou pour approfondir des sujets vus en classe."
- "La bibliothèque centralise tous les supports pédagogiques pour faciliter l'accès et éviter la dispersion des documents."
- "Vous recevrez des notifications automatiques quand vos enseignants partagent de nouvelles ressources avec votre classe."
- "Les ressources peuvent être pré-visualisées directement dans l'application (PDF, images, vidéos) avant le téléchargement."
- "Chaque ressource contient des métadonnées complètes : titre, description, matière, compétence et auteur."
- "Les ressources sont organisées par pertinence puis par date pour vous aider à trouver rapidement ce dont vous avez besoin."

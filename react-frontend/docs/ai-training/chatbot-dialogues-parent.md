# Dialogues Chatbot - Parents

## Vue d'ensemble

Ce document contient des exemples de dialogues entre parents et le chatbot EdConnekt, basés sur les fonctionnalités identifiées dans le code source.

## Intentions et Réponses

### Suivi des enfants

**Exemples de phrases utilisateur :**
- "Comment suivre mes enfants ?"
- "Je veux voir les résultats de mon enfant"
- "Comment sélectionner un enfant ?"
- "Où voir le dashboard de mon enfant ?"
- "Comment changer d'enfant ?"
- "Où voir les informations de mes enfants ?"

**Réponses attendues du chatbot :**
- "Connectez-vous et sélectionnez un enfant dans la liste pour consulter son dashboard avec vue d'ensemble de ses résultats et activités."
- "Dans votre espace parent, vous pouvez sélectionner un enfant et consulter son dashboard avec toutes ses informations scolaires."
- "Le dashboard de votre enfant vous donne un aperçu complet de ses cours, notes, événements et progression."
- "Utilisez le sélecteur d'enfant en haut de l'écran pour naviguer entre vos différents enfants et consulter leurs informations."

### Consultation des notes

**Exemples de phrases utilisateur :**
- "Comment voir les notes de mon enfant ?"
- "Où consulter les résultats scolaires ?"
- "Je veux voir les évaluations"
- "Comment analyser la progression ?"
- "Où voir les moyennes par matière ?"
- "Comment comparer les trimestres ?"
- "Comment comprendre les différents types d'évaluation ?"
- "Comment voir les notes continues de mon enfant ?"
- "Comment voir les notes d'intégration ?"
- "Comment voir les notes trimestrielles ?"
- "Comment analyser la progression de mon enfant ?"
- "Comment voir les moyennes par domaine ?"
- "Comment télécharger les rapports de notes ?"
- "Comment comparer les résultats entre trimestres ?"
- "Comment voir les points forts et faibles ?"
- "Comment comprendre les pourcentages ?"
- "Comment voir l'historique des notes ?"
- "Comment changer d'enfant pour voir ses notes ?"

**Réponses attendues du chatbot :**
- "Allez dans 'Notes' depuis le menu, sélectionnez l'enfant et la période, puis consultez les résultats par matière avec graphiques de progression."
- "Dans 'Notes', vous pouvez choisir la période d'évaluation et analyser la progression de votre enfant via les graphiques."
- "Consultez la section 'Notes' pour voir les évaluations par matière et télécharger les rapports si nécessaire."
- "Les notes sont organisées par matière avec des graphiques de progression qui vous permettent de suivre l'évolution de votre enfant."
- "Trois types d'évaluation sont disponibles : Continue (évaluations régulières), Intégration (évaluations de synthèse), Trimestrielle (bilans périodiques)."
- "Les notes continues montrent la progression régulière de votre enfant tout au long de l'année sur chaque compétence."
- "Les notes d'intégration évaluent la maîtrise globale des compétences acquises par votre enfant dans une matière."
- "Les notes trimestrielles fournissent un bilan complet avec moyennes et recommandations pour soutenir votre enfant."
- "Analysez la progression via les graphiques qui montrent l'évolution des compétences de votre enfant dans le temps."
- "Les moyennes sont calculées par domaine (Langues, Sciences, etc.) et par matière pour un suivi complet."
- "Téléchargez les rapports de notes en PDF depuis la barre d'outils pour les conserver ou les partager."
- "Comparez les résultats entre les différents trimestres pour identifier les progrès et axes d'amélioration."
- "Les points forts et faibles sont identifiés automatiquement avec des indicateurs visuels (vert = acquis, orange = en cours, rouge = à travailler)."
- "Les pourcentages indiquent le niveau d'acquisition de chaque compétence : au-dessus de 70% = acquis, en dessous = à renforcer."
- "L'historique des notes est conservé pour vous permettre de suivre la progression de votre enfant sur l'année entière."
- "Utilisez le sélecteur d'enfant en haut de l'écran pour naviguer entre vos différents enfants et consulter leurs notes respectives."

- "Les ressources partagées par les enseignants sont accessibles dans l'onglet 'Ressources' de chaque remédiation avec possibilité de téléchargement."
- "Les documents de remédiation incluent des exercices, des supports de cours et des méthodes pédagogiques que vous pouvez consulter et télécharger."
- "Les ressources disponibles sont organisées par type (PDF, vidéos, exercices) et vous pouvez les utiliser pour soutenir votre enfant à la maison."
- "Téléchargez les supports de remédiation depuis la section 'Ressources' pour avoir accès aux documents même hors ligne."
- "Les méthodes peuvent être individuelles (travail personnalisé), de groupe (collaboration) ou mixtes selon les objectifs de la remédiation."
- "Chaque méthode pédagogique est détaillée avec sa description, sa durée et son type d'intervention pour vous aider à comprendre l'approche utilisée."

# Procédures Utilisateurs EdConnekt

## Procédures pour les Élèves

### Connexion et première utilisation
1. **Accès à la plateforme** : Aller sur l'URL d'EdConnekt
2. **Connexion** : Utiliser identifiant et mot de passe fournis
3. **Première connexion** : Changer le mot de passe par défaut
4. **Vérification du profil** : Contrôler les informations personnelles

### Consultation des cours
1. **Accéder à "Mes cours"** depuis le menu principal
2. **Sélectionner une matière** dans la liste
3. **Consulter les leçons** disponibles
4. **Télécharger les ressources** associées
5. **Marquer comme terminé** si applicable

### Suivi des notes
1. **Accéder à "Mes notes"** depuis le menu
2. **Sélectionner la période** (trimestre, mois)
3. **Consulter les évaluations** par matière
4. **Analyser la progression** via les graphiques
5. **Identifier les points d'amélioration**

### Utilisation de l'agenda
1. **Accéder à "Agenda"** depuis le menu
2. **Consulter les événements** du jour/semaine
3. **Voir les remédiations** programmées
4. **Noter les devoirs** à rendre
5. **Synchroniser** avec le calendrier personnel

### Communication
1. **Accéder à "Messages"** depuis le menu
2. **Envoyer un message** à un enseignant
3. **Consulter les réponses** reçues
4. **Joindre des documents** si nécessaire
5. **Marquer comme lu** les messages consultés

## Procédures pour les Enseignants

### Gestion des classes
1. **Accéder à "Élèves"** depuis le menu principal
2. **Sélectionner une classe** dans la liste
3. **Consulter la liste des élèves** avec photos
4. **Vérifier les présences** quotidiennes
5. **Accéder aux profils individuels** pour plus de détails

### Création de cours
1. **Accéder à "Cours"** depuis le menu
2. **Cliquer sur "Créer un cours"**
3. **Remplir les informations** : titre, matière, classe
4. **Ajouter des leçons** avec descriptions
5. **Joindre des ressources** (PDF, vidéos, exercices)
6. **Publier le cours** pour les élèves

### Saisie des notes
1. **Accéder à "Évaluations"** depuis le menu
2. **Sélectionner la classe** et la matière
3. **Choisir le type d'évaluation** (devoir, contrôle, oral)
4. **Saisir les notes** par élève et compétence
5. **Ajouter des commentaires** si nécessaire
6. **Valider et publier** les résultats

### Organisation des remédiations
1. **Accéder à "Remédiations"** depuis le menu
2. **Identifier les élèves** en difficulté
3. **Créer une session** de remédiation
4. **Sélectionner les ressources** adaptées
5. **Programmer la session** (date, heure, durée)
6. **Notifier les élèves** et parents concernés

### Suivi PDI
1. **Accéder à "PDI"** depuis le menu
2. **Consulter les élèves** en projet d'intégration
3. **Évaluer les compétences** spécifiques
4. **Générer les rapports** de progression
5. **Communiquer avec les parents** si nécessaire

## Procédures pour les Parents

### Suivi des enfants
1. **Se connecter** avec les identifiants fournis
2. **Sélectionner un enfant** dans la liste
3. **Consulter le dashboard** avec vue d'ensemble
4. **Accéder aux détails** par section

### Consultation des notes
1. **Accéder à "Notes"** depuis le menu
2. **Sélectionner l'enfant** concerné
3. **Choisir la période** d'évaluation
4. **Consulter les résultats** par matière
5. **Analyser la progression** via graphiques
6. **Télécharger les rapports** si nécessaire

### Communication avec l'équipe
1. **Accéder à "Messages"** depuis le menu
2. **Sélectionner le destinataire** (enseignant, direction)
3. **Rédiger le message** avec objet clair
4. **Joindre des documents** si nécessaire
5. **Envoyer et attendre** la réponse
6. **Consulter l'historique** des échanges

### Suivi des événements
1. **Accéder à "Événements"** depuis le menu
2. **Consulter le calendrier** scolaire
3. **Noter les dates importantes** (réunions, sorties)
4. **S'inscrire** aux événements si demandé
5. **Recevoir les notifications** automatiques

### Consultation des rapports
1. **Accéder à "Rapports"** depuis le menu
2. **Sélectionner l'enfant** concerné
3. **Choisir le type de rapport** (trimestriel, remédiation)
4. **Consulter les détails** et recommandations
5. **Télécharger le PDF** si disponible
6. **Prendre rendez-vous** si nécessaire

## Procédures pour les Administrateurs

### Gestion des utilisateurs
1. **Accéder à "Utilisateurs"** depuis le menu
2. **Créer un nouveau compte** avec rôle approprié
3. **Attribuer les permissions** selon le profil
4. **Vérifier les accès** et droits
5. **Gérer les mots de passe** oubliés

### Configuration des classes
1. **Accéder à "Classes"** depuis le menu
2. **Créer une nouvelle classe** avec effectif
3. **Assigner les enseignants** responsables
4. **Configurer les matières** enseignées
5. **Définir les horaires** et emplois du temps

### Gestion des ressources
1. **Accéder à "Ressources"** depuis le menu
2. **Uploader des documents** (PDF, vidéos, exercices)
3. **Catégoriser les ressources** par matière/niveau
4. **Partager avec les enseignants** concernés
5. **Gérer les permissions** d'accès

### Génération de rapports
1. **Accéder à "Rapports"** depuis le menu
2. **Sélectionner le type** de rapport souhaité
3. **Définir les paramètres** (période, classe, matière)
4. **Générer le rapport** en PDF
5. **Partager avec les destinataires** concernés

### Maintenance système
1. **Accéder à "Configuration"** depuis le menu
2. **Vérifier les sauvegardes** automatiques
3. **Mettre à jour les données** si nécessaire
4. **Gérer les notifications** système
5. **Contrôler les performances** de la plateforme

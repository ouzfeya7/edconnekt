# Cas d'Usage Types EdConnekt

## Cas d'usage pour les Élèves

### Scénario 1 : Consultation des notes
**Contexte :** Un élève veut consulter ses résultats du dernier trimestre
**Actions :**
1. Se connecter à EdConnekt
2. Aller dans "Mes notes" depuis le menu
3. Sélectionner "Trimestre 2" dans le filtre
4. Consulter les résultats par matière
5. Analyser les graphiques de progression
6. Identifier les matières à améliorer

### Scénario 2 : Préparation d'un cours
**Contexte :** Un élève veut réviser avant un cours de mathématiques
**Actions :**
1. Accéder à "Mes cours" depuis le menu
2. Sélectionner "Mathématiques" dans la liste
3. Consulter la leçon "Résoudre une équation du second degré"
4. Télécharger les ressources associées
5. Marquer la leçon comme consultée

### Scénario 3 : Communication avec un enseignant
**Contexte :** Un élève a une question sur un devoir
**Actions :**
1. Aller dans "Messages" depuis le menu
2. Sélectionner l'enseignant de mathématiques
3. Rédiger le message : "Bonjour, j'ai une question sur l'exercice 3 du devoir de la semaine dernière"
4. Joindre une photo du travail si nécessaire
5. Envoyer le message et attendre la réponse

### Scénario 4 : Consultation de l'agenda
**Contexte :** Un élève veut voir ses activités de la semaine
**Actions :**
1. Accéder à "Agenda" depuis le menu
2. Consulter les événements de la semaine
3. Noter les remédiations programmées
4. Vérifier les devoirs à rendre
5. Synchroniser avec le calendrier personnel

## Cas d'usage pour les Enseignants

### Scénario 1 : Gestion de classe quotidienne
**Contexte :** Un enseignant veut faire l'appel et consulter les présences
**Actions :**
1. Se connecter à EdConnekt
2. Aller dans "Élèves" depuis le menu
3. Sélectionner sa classe (ex: 6ème A)
4. Consulter la liste des élèves avec photos
5. Marquer les présences/absences
6. Noter les justificatifs si nécessaire

### Scénario 2 : Création d'un nouveau cours
**Contexte :** Un enseignant veut créer un cours sur la géométrie
**Actions :**
1. Accéder à "Cours" depuis le menu
2. Cliquer sur "Créer un cours"
3. Remplir les informations : "Géométrie plane", "Mathématiques", "6ème A"
4. Ajouter des leçons : "Les triangles", "Le cercle", "Les angles"
5. Joindre des ressources : exercices PDF, vidéos explicatives
6. Publier le cours pour les élèves

### Scénario 3 : Saisie des notes
**Contexte :** Un enseignant doit saisir les résultats d'un contrôle
**Actions :**
1. Aller dans "Évaluations" depuis le menu
2. Sélectionner la classe et la matière
3. Choisir "Contrôle de géométrie" comme type d'évaluation
4. Saisir les notes par élève et compétence :
   - Compétence 1 : Reconnaître les figures (sur 10)
   - Compétence 2 : Calculer les périmètres (sur 10)
5. Ajouter des commentaires pour certains élèves
6. Valider et publier les résultats

### Scénario 4 : Organisation d'une remédiation
**Contexte :** Un enseignant constate que plusieurs élèves ont des difficultés en géométrie
**Actions :**
1. Accéder à "Remédiations" depuis le menu
2. Identifier les élèves en difficulté (notes < 10/20)
3. Créer une session "Remédiation géométrie"
4. Sélectionner les ressources adaptées : exercices progressifs, vidéos
5. Programmer la session : vendredi 14h-15h, salle 12
6. Notifier automatiquement les élèves et parents concernés

### Scénario 5 : Suivi PDI
**Contexte :** Un enseignant doit évaluer un élève en projet d'intégration
**Actions :**
1. Aller dans "PDI" depuis le menu
2. Sélectionner l'élève concerné
3. Évaluer les compétences spécifiques :
   - Langage : 7/10
   - Conte : 8/10
   - Vocabulaire : 6/10
   - Lecture : 7/10
   - Graphisme : 8/10
4. Ajouter des remarques sur la progression
5. Générer le rapport de suivi

## Cas d'usage pour les Parents

### Scénario 1 : Suivi des résultats d'un enfant
**Contexte :** Un parent veut consulter les notes de son enfant
**Actions :**
1. Se connecter à EdConnekt
2. Sélectionner l'enfant dans la liste
3. Aller dans "Notes" depuis le menu
4. Choisir "Trimestre 2" dans le filtre
5. Consulter les résultats par matière avec graphiques
6. Analyser la progression depuis le trimestre précédent

### Scénario 2 : Communication avec l'équipe enseignante
**Contexte :** Un parent veut informer d'une absence prévue
**Actions :**
1. Accéder à "Messages" depuis le menu
2. Sélectionner l'enseignant principal ou la direction
3. Rédiger le message : "Mon enfant sera absent le 15 mars pour un rendez-vous médical"
4. Joindre un justificatif si nécessaire
5. Envoyer le message et attendre l'accusé de réception

### Scénario 3 : Consultation des événements scolaires
**Contexte :** Un parent veut voir le calendrier des activités
**Actions :**
1. Aller dans "Événements" depuis le menu
2. Consulter le calendrier scolaire du mois
3. Noter les dates importantes : réunion parents-professeurs, sortie scolaire
4. S'inscrire aux événements si demandé
5. Recevoir les notifications automatiques

### Scénario 4 : Consultation d'un rapport de remédiation
**Contexte :** Un parent reçoit une notification de remédiation
**Actions :**
1. Ouvrir la notification reçue
2. Accéder au rapport de remédiation
3. Consulter les détails : compétences travaillées, progression
4. Lire les recommandations de l'enseignant
5. Télécharger le PDF du rapport si nécessaire

## Cas d'usage pour les Administrateurs

### Scénario 1 : Création d'un compte utilisateur
**Contexte :** Un nouvel enseignant arrive dans l'établissement
**Actions :**
1. Se connecter en tant qu'administrateur
2. Aller dans "Utilisateurs" depuis le menu
3. Cliquer sur "Créer un compte"
4. Remplir les informations : nom, prénom, email, rôle "enseignant"
5. Attribuer les permissions : accès aux classes, saisie de notes
6. Envoyer les identifiants par email

### Scénario 2 : Configuration d'une nouvelle classe
**Contexte :** Une nouvelle classe de 5ème est créée
**Actions :**
1. Accéder à "Classes" depuis le menu
2. Cliquer sur "Créer une classe"
3. Remplir les informations : "5ème B", effectif 28 élèves
4. Assigner les enseignants responsables par matière
5. Configurer l'emploi du temps
6. Créer les comptes élèves automatiquement

### Scénario 3 : Gestion des ressources pédagogiques
**Contexte :** L'établissement reçoit de nouveaux manuels numériques
**Actions :**
1. Aller dans "Ressources" depuis le menu
2. Uploader les fichiers PDF des manuels
3. Catégoriser par matière et niveau
4. Partager avec les enseignants concernés
5. Définir les permissions d'accès pour les élèves

### Scénario 4 : Génération d'un rapport trimestriel
**Contexte :** Fin du trimestre, génération des bilans
**Actions :**
1. Accéder à "Rapports" depuis le menu
2. Sélectionner "Rapport trimestriel"
3. Définir les paramètres : classe, période, matières
4. Générer le rapport en PDF
5. Envoyer automatiquement aux parents concernés

## Cas d'usage transversaux

### Scénario : Gestion d'une absence d'élève
**Contexte :** Un élève est absent, notification aux parents
**Actions :**
1. **Enseignant** : Marquer l'absence dans la classe
2. **Système** : Générer automatiquement une notification
3. **Parent** : Recevoir l'alerte et justifier si nécessaire
4. **Administration** : Suivre les absences répétées
5. **Communication** : Échange de messages si besoin

### Scénario : Suivi d'une remédiation
**Contexte :** Un élève participe à une session de remédiation
**Actions :**
1. **Enseignant** : Organiser la session et sélectionner les ressources
2. **Système** : Notifier l'élève et les parents
3. **Élève** : Participer à la session et consulter les ressources
4. **Enseignant** : Évaluer la progression et saisir les résultats
5. **Parent** : Consulter le rapport de remédiation
6. **Administration** : Suivre l'efficacité des remédiations

### Scénario : Communication parents-enseignants
**Contexte :** Un parent s'inquiète des résultats de son enfant
**Actions :**
1. **Parent** : Envoyer un message à l'enseignant
2. **Enseignant** : Consulter le profil de l'élève et répondre
3. **Système** : Notifier le parent de la réponse
4. **Suivi** : Échange de messages si nécessaire
5. **Action** : Organisation d'une remédiation si besoin

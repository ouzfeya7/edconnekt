# FAQ Générales EdConnekt

## Questions d'accès et connexion

### Comment me connecter à EdConnekt ?
R : Utilisez votre identifiant et mot de passe fournis par l'établissement. Si c'est votre première connexion, vous devrez changer le mot de passe par défaut.

### J'ai oublié mon mot de passe, que faire ?
R : Contactez l'administrateur de votre établissement ou utilisez la fonction "Mot de passe oublié" sur la page de connexion.

### Je ne peux pas accéder à certaines fonctionnalités, pourquoi ?
R : Vérifiez que vous êtes connecté avec le bon rôle (élève, enseignant, parent, administrateur). Chaque rôle a des permissions spécifiques.

### La plateforme est-elle accessible sur mobile ?
R : Oui, EdConnekt est responsive et s'adapte aux smartphones et tablettes. Toutes les fonctionnalités principales sont disponibles sur mobile.

## Questions pour les Élèves

### Comment consulter mes notes ?
R : Connectez-vous, allez dans "Mes notes" depuis le menu principal, sélectionnez la période souhaitée et consultez vos résultats par matière.

### Où trouver mes cours et leçons ?
R : Accédez à "Mes cours" depuis le menu, sélectionnez une matière et consultez les leçons disponibles avec leurs ressources associées.

### Comment utiliser l'agenda ?
R : Allez dans "Agenda" pour voir vos cours, événements, remédiations et devoirs. Vous pouvez synchroniser avec votre calendrier personnel.

### Comment contacter un enseignant ?
R : Utilisez la section "Messages" pour envoyer un message à un enseignant. Vous pouvez joindre des documents si nécessaire.

### Comment voir ma progression ?
R : Dans "Mes notes", vous trouverez des graphiques de progression par compétence et par matière, avec des indicateurs visuels de votre évolution.

## Questions pour les Enseignants

### Comment gérer ma classe ?
R : Accédez à "Élèves" depuis le menu, sélectionnez votre classe et consultez la liste des élèves avec leurs informations et présences.

### Comment créer un cours ?
R : Allez dans "Cours", cliquez sur "Créer un cours", remplissez les informations (titre, matière, classe) et ajoutez des leçons avec ressources.

### Comment saisir les notes ?
R : Dans "Évaluations", sélectionnez la classe et matière, choisissez le type d'évaluation et saisissez les notes par élève et compétence.

### Comment organiser une remédiation ?
R : Accédez à "Remédiations", identifiez les élèves en difficulté, créez une session avec ressources adaptées et programmez-la.

### Comment suivre les PDI ?
R : Utilisez la section "PDI" pour évaluer les compétences spécifiques des élèves en projet d'intégration et générer des rapports.

## Questions pour les Parents

### Comment suivre mes enfants ?
R : Connectez-vous, sélectionnez un enfant dans la liste et consultez son dashboard avec vue d'ensemble de ses résultats et activités.

### Comment voir les notes de mon enfant ?
R : Allez dans "Notes", sélectionnez l'enfant et la période, consultez les résultats par matière avec graphiques de progression.

### Comment contacter l'équipe enseignante ?
R : Utilisez "Messages" pour communiquer avec les enseignants ou la direction. Vous pouvez joindre des documents si nécessaire.

### Comment être informé des événements ?
R : Consultez "Événements" pour voir le calendrier scolaire, les réunions, sorties et activités. Vous recevrez des notifications automatiques.

### Comment consulter les rapports ?
R : Dans "Rapports", sélectionnez l'enfant et le type de rapport (trimestriel, remédiation) pour voir les détails et recommandations.

## Questions pour les Administrateurs

### Comment créer un compte utilisateur ?
R : Accédez à "Utilisateurs", cliquez sur "Créer un compte", remplissez les informations et attribuez le rôle approprié avec les permissions.

### Comment configurer une classe ?
R : Dans "Classes", créez une nouvelle classe avec l'effectif, assignez les enseignants responsables et configurez les matières enseignées.

### Comment gérer les ressources ?
R : Utilisez "Ressources" pour uploader des documents, les catégoriser par matière/niveau et partager avec les enseignants concernés.

### Comment générer des rapports ?
R : Allez dans "Rapports", sélectionnez le type souhaité, définissez les paramètres (période, classe, matière) et générez le PDF.

## Questions techniques

### La plateforme est-elle sécurisée ?
R : Oui, EdConnekt utilise des protocoles de sécurité standards avec authentification sécurisée et chiffrement des données.

### Mes données sont-elles sauvegardées ?
R : Oui, des sauvegardes automatiques sont effectuées régulièrement pour protéger toutes les données de la plateforme.

### Comment signaler un problème technique ?
R : Contactez l'administrateur de votre établissement ou utilisez la fonction "Support" intégrée dans la plateforme.

### La plateforme fonctionne-t-elle hors ligne ?
R : Non, EdConnekt nécessite une connexion internet pour fonctionner correctement et synchroniser les données en temps réel.

## Questions sur les fonctionnalités

### Qu'est-ce que le PDI ?
R : Le PDI (Projet d'Intégration) est un suivi spécifique pour certains élèves avec évaluations périodiques de compétences particulières.

### Comment fonctionnent les remédiations ?
R : Les remédiations sont des sessions de soutien personnalisées organisées par les enseignants pour aider les élèves en difficulté.

### Que sont les compétences ?
R : Les compétences sont les savoirs évalués par domaine (français, mathématiques, etc.) avec progression trimestrielle et indicateurs d'acquisition.

### Comment fonctionne le système de notes ?
R : Les notes sont saisies par compétence et matière, avec possibilité d'ajouter des commentaires et de suivre la progression dans le temps.

## Questions sur la communication

### Comment recevoir des notifications ?
R : Les notifications apparaissent automatiquement dans l'interface et peuvent être configurées selon vos préférences dans les paramètres.

### Comment joindre des documents aux messages ?
R : Lors de la rédaction d'un message, utilisez le bouton "Joindre un fichier" pour sélectionner et envoyer des documents.

### Les messages sont-ils privés ?
R : Oui, les messages sont privés entre l'expéditeur et le destinataire, avec historique conservé pour consultation.

### Comment savoir si un message a été lu ?
R : Les messages consultés sont automatiquement marqués comme "lu" et vous pouvez voir le statut dans votre boîte de réception.
